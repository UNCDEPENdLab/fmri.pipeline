# internal function to populate l1_model_set onsets from spec specification

internal function to populate l1_model_set onsets from spec
specification

## Usage

``` r
fields_from_spec(l1_model_set, slist, trial_data, field_names = NULL)
```

## Arguments

- l1_model_set:

  An L1 model set.

- slist:

  Parsed signal/model specification list.

- trial_data:

  Trial-level data.

- field_names:

  Fields to populate from the specification.
