# helper function to read confounds and motion parameters and combine them

helper function to read confounds and motion parameters and combine them

## Usage

``` r
read_confounds_motion_parameters(
  gpa,
  id,
  session,
  run_number,
  run_df,
  expected_l1_confound_file,
  lg,
  confound_path = NA_character_,
  motion_path = NA_character_
)
```
