# helper function for setting up 3dClustSim objects for permutation (via 3dttest++).

helper function for setting up 3dClustSim objects for permutation (via
3dttest++).

## Usage

``` r
build_3dclustsim_permutation(to_fwe)
```

## Arguments

- to_fwe:

  a data.frame containing model information for the group analyses to be
  passed through 3dClustSim FWE
