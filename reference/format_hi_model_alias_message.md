# Format a clear diagnostic for aliased higher-level model coefficients

Format a clear diagnostic for aliased higher-level model coefficients

## Usage

``` r
format_hi_model_alias_message(mobj, model_formula = NULL)
```

## Arguments

- mobj:

  a model object populated by `mobj_fit_lm`

- model_formula:

  optional formula string used for display

## Value

a character string suitable for an error or interactive prompt
