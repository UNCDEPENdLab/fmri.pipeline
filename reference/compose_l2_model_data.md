# Build run-level L2 model data augmented with session-level predictors

Build run-level L2 model data augmented with session-level predictors

## Usage

``` r
compose_l2_model_data(gpa, lg = NULL)
```

## Arguments

- gpa:

  A `glm_pipeline_arguments` object.

- lg:

  Optional logger.

## Value

A data.frame suitable for L2 model fitting/respecification.
