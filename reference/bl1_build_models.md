# helper function to create a set of level 1 models

helper function to create a set of level 1 models

## Usage

``` r
bl1_build_models(l1_model_set, spec_list = NULL, lg = NULL)
```

## Arguments

- l1_model_set:

  a set of level 1 models

- spec_list:

  if creating models from a spec (YAML) file, this is the parsed list

- lg:

  the current logger
