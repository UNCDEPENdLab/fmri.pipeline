# Small helper function to calculate collinearity diagnostics on the events in the design prior to convolution. This can help to diagnose problems with parametric modulators that correlate excessively, for example.

Small helper function to calculate collinearity diagnostics on the
events in the design prior to convolution. This can help to diagnose
problems with parametric modulators that correlate excessively, for
example.

## Usage

``` r
get_collin_events(dmat)
```

## Arguments

- dmat:

  A design matrix list generated by `build_design_matrix` that contains
  a set of regressors for each run, where run is indicated by the rows
  of `dmat`
