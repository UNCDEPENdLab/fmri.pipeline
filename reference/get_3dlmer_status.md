# Status checker for 3dLMEr

Status checker for 3dLMEr

## Usage

``` r
get_3dlmer_status(output_file, lg = NULL, prefix = NULL)
```

## Arguments

- output_file:

  expected output file from 3dLMEr

- lg:

  optional logger

- prefix:

  optional prefix for returned column names

## Value

data.frame indicating output existence and completion status
